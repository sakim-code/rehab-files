# 재활치료클리닉 자료실

상담자료와 이미지를 웹에서 바로 보고 다운로드할 수 있는 사이트입니다.
`images` 폴더와 `documents` 폴더에 파일만 올리면 사이트에 자동으로 나타납니다.

## 폴더 구조
- `index.html` — 사이트 화면 (설정은 이 파일 위쪽 CONFIG 부분에서)
- `images/` — 이미지 파일을 올리는 곳 (하위 폴더를 만들면 카테고리로 표시)
- `documents/` — PDF·문서 등 상담자료를 올리는 곳 (하위 폴더 = 카테고리)

## 파일 올리는 법
1. 올리려는 폴더(`images` 또는 `documents`)를 클릭
2. 오른쪽 위 **Add file → Upload files** 클릭
3. 파일을 끌어다 놓고 아래 **Commit changes** 클릭
4. 1~2분 뒤 사이트를 새로고침하면 반영됩니다.
